# متجر الإلكترونيات

هذا المشروع هو متجر إلكترونيات تم بناؤه باستخدام **React** و **Vite** و **Tailwind CSS**.  
الموقع يحتوي على:

- الصفحة الرئيسية مع Hero, Featured Products, Special Offers, Newsletter, Why Us, Testimonials.
- صفحات المنتجات، تفاصيل المنتج، من نحن، اتصل بنا، الدفع، الخصوصية، الشروط والأحكام.
- مكونات UI مثل Button, Input, Badge.
- نظام تنقل (Router) بين الصفحات.
